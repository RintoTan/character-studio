# 🌟 Overview | 项目简介

**Character Studio** 是一个专为原创角色（OC）设计的角色管理工具。

Character Studio is a modern workspace for creating, editing, organizing and exporting original characters.

核心理念：

- 创作优先（Creation First）
- 信息集中（Everything in One Place）
- 简洁高效（Simple & Efficient）
- 本地优先（Local First）
- 持续演进（Continuously Improving）
